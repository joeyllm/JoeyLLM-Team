# 🌐 JoeyLLM Platforms & Repositories

Welcome to the **JoeyLLM** workspace. This directory outlines the official platforms, repositories, and communication tools our team uses to build, track, and publish our work.

To keep a complex project like a Large Language Model organized, we have to be extremely disciplined about where our code lives, how we communicate, and how we share our results with the world. 

---

## 🛑 Our Core Philosophy

1. **Consolidate Everything:** We do not fragment our workflow. If a tool isn't listed in this directory, we don't use it. (e.g., We use GitHub for project management, not external tools like Trello or Jira).
2. **Be Clean & Kind:** When interacting on public platforms (like Hugging Face or Kaggle) or welcoming new contributors on Discord, you represent the JoeyLLM team. Always maintain a highly professional, well-documented, and welcoming presence.

---

## 📚 Platform Index

Please read through each of the documents below to understand the rules and expectations for our project toolkit. Make sure you have accounts created for all of the necessary platforms.

| Platform / File | Purpose |
| :--- | :--- |
| [**Discord**](Discord.md) | **(Mandatory)** Our central communication hub for daily syncs, team meetings, and welcoming open-source contributors. |
| [**GitHub**](GitHub.md) | **(Mandatory)** The absolute source of truth for all pipeline code, version control, and project tracking/Kanban boards. |
| [**Weights & Biases (W&B)**](WeightsAndBiases.md) | **(Mandatory)** Our dashboard for tracking ML experiments, loss curves, and GPU metrics. Start learning this now before Term 2! |
| [**Hugging Face**](HuggingFace.md) | Where we publish our final models, data cards, and cleaned 60TB dataset to the global AI community. |
| [**Kaggle**](Kaggle.md) | Our exploratory space for hosting dataset samples, sharing EDA notebooks, and learning the Kaggle community culture. |
| [**DockerHub**](DockerHub.md) | One of our amazing project sponsors! However, note that we are experimenting with GitHub Actions for container builds this term. |
